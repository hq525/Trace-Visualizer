# crashpath

Paste a raw stack trace → see the failure as an animated path through your
codebase graph. Post-mortem, local-first, zero setup: `npx crashpath`, no API
key, no Docker, works offline.

**v0.1 is in active development** — the Python end-to-end pipeline (trace
extraction, tree-sitter symbol resolution, interactive spine UI) is built and
tested; JS/TS support, ghost edges, and exports are landing next.

This version reserves the package name. Follow the first working release at
<https://github.com/hq525/Trace-Visualizer>.
